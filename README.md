# これはなに #

base64 で変換された文字列を元に戻す機能をもったアドオンです。


# インストール #

(use Addon-SDK)
`$ cfx xpi`


# 使い方 #

1. base64 で変換された文字列をドラッグして選択する
2. 右クリックして出てきたメニューの「decode as base64」をクリック


# 使用したライブラリ #

* js-base64 <https://github.com/dankogai/js-base64>
